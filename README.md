# Smart Inventory & Sales Management System

## Overview

Smart Inventory & Sales Management System is a full-stack web application developed to help businesses manage inventory, sales, customers, and reports efficiently. The system provides real-time stock tracking, sales monitoring, and analytical reporting to improve business operations.

## Features

### Inventory Management
- Add, update, and delete products
- Manage product categories
- Track stock quantities
- Low stock alerts
- Product search and filtering

### Sales Management
- Create sales invoices
- Record transactions
- Sales history tracking
- Revenue monitoring

### Customer Management
- Customer registration
- Customer information management
- Purchase history tracking

### Dashboard
- Total Products
- Total Sales
- Revenue Statistics
- Low Stock Products
- Business Insights

### Reports
- Sales Reports
- Inventory Reports
- Customer Reports
- Revenue Analysis

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/smart-inventory-management-system.git
```

### Install Dependencies

```bash
npm install
```

### Start Backend

```bash
npm run server
```

### Start Frontend

```bash
npm start
```
